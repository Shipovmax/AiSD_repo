## Задача 6. Проходимость лабиринта

Вам дан список строк, представляющий лабиринт:

```
maze = [
  'S----',
  '##---',
  '---##',
  '----X'
]
```

**Обозначения:**
- `S` — стартовая позиция
- `X` — финиш
- `-` — свободный путь
- `#` — стена (непроходима)

**Правила:**
- Игрок может двигаться только на 1 шаг за раз
- Доступные направления: вверх, вниз, влево, вправо

Напишите рекурсивную функцию, которая принимает лабиринт и возвращает `True`, если он проходим, и `False` в любом другом случае.

---

## Задача 7. Решение лабиринта

Используя тот же лабиринт из задачи 6, напишите рекурсивную функцию, которая возвращает решение — список шагов, позволяющих игроку добраться до финиша.

```python
solve(maze)  # ['right', 'right', 'down', 'down', 'down', 'right', 'right']
```

Решение не обязательно должно быть кратчайшим. Если лабиринт непроходим — вернуть `[]`.
